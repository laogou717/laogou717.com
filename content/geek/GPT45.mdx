## 一、GPT-4.5 概述

OpenAI 最近发布了 GPT-4.5 模型，这是继 GPT-4o 之后的又一次更新。根据 Sam Altman 的说法，GPT-4.5 在以下几个方面有所提升：

- 更擅长写作和编程
- 增强了创意性工作能力
- 更加"有人味"，能更精准理解用户情绪

然而，这些提升是否真的如宣传所说那样显著？本文将通过实际体验，为大家带来一次深度分析。

## 二、价格分析：昂贵的体验

GPT-4.5 的价格设定令人咋舌：

- 输入：75 美元/百万 token
- 输出：150 美元/百万 token

为了让大家有一个直观的概念，我们可以做个简单对比：

- 在 DeepSeek 平台上，10 元人民币可以购买 500 万 token
- 而 GPT-4.5 的 100 万 token 输入就需要 75 美元

这意味着在没有命中缓存的情况下，让 GPT-4.5 输出约 1,000 字的内容，大约会消耗 4-5 元人民币。对于个人用户来说，这个价格确实不太友好。

## 三、新功能体验：应用控制能力

### 3.1 文本编辑器集成

GPT-4.5 最值得关注的新功能是其应用控制能力。现在，ChatGPT 可以直接在文本编辑器中写入内容：

1. 按下快捷键呼出对话框
2. 选中文本编辑器
3. 输入指令（如"写一段 AE 表达式"）
4. ChatGPT 会直接在编辑器中写入代码

这项功能支持主流编辑器，包括：
- VS Code
- Xcode
- Cursor

### 3.2 设置与配置

要使用这一功能，需要在 ChatGPT 桌面端进行设置：

1. 在设置中向下滑动找到"管理应用"
2. 点击后可以看到可操作的应用列表
3. 对于 VS Code，需要安装扩展程序
4. 安装完成后，VS Code 就会变为"可用"状态

### 3.3 实际应用案例

在实际测试中，我尝试让 ChatGPT 在一个静态网页中添加鼠标点击动画（彩带飘落效果）。虽然 ChatGPT 成功编写了代码并添加到页面中，但效果并不理想，显示出这一功能仍有改进空间。

## 四、高级语音模式下的应用控制

在高级语音模式下，GPT-4.5 也可以与应用进行交互：

### 4.1 备忘录交互

通过高级语音模式，可以让 ChatGPT 查看备忘录内容：

- 在高级语音模式下点击"控制应用"
- 选择备忘录
- 询问"你在我的备忘录里看到了什么"
- ChatGPT 会读取并回答备忘录内容

### 4.2 代码查看与解释

同样，ChatGPT 也可以查看和解释 VS Code 中的代码：

- 询问"你现在看一下我 VS Code 中这行代码能看到什么"
- ChatGPT 会解释代码的功能和结构
- 还可以询问特定单词的发音（如"Segoe UI"）

## 五、功能局限与未来展望

### 5.1 当前局限

尽管这一功能有潜力，但目前仍存在明显局限：

- 高级语音聊天模式下无法进行编辑，只能查看
- 只能通过文字输入模式实时编辑
- 功能整体感觉像是"半成品"，距离完善还有距离

### 5.2 未来可能的发展方向

这一功能未来可能的发展方向包括：

- 接入更多应用，如 Photoshop、Premiere Pro、Final Cut Pro X、剪映等
- 在剪辑软件中提供效果实现指导
- 添加鼠标聚焦功能，指导用户操作界面
- 在高级语音模式下实现编辑功能

## 六、总结

GPT-4.5 的应用控制功能展示了 AI 与日常工作流程深度融合的潜力，但目前仍处于早期阶段。虽然价格昂贵且功能有限，但这无疑是 AI 辅助工作的一个重要方向。

随着功能的不断完善和价格的可能调整，我们可以期待这一功能在未来为内容创作、编程和日常工作带来更多便利。目前，它更像是一个有趣的技术预览，而非必备的生产力工具。

对于关注 AI 发展的用户来说，值得关注这一功能的后续更新，但可能不必急于为此支付高昂的使用费用。
